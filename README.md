## Overview

AnaliseATS is an AI-powered web application that evaluates how well a resume matches a specific job posting.

Instead of simply checking formatting, the platform analyses the relationship between the job description and the candidate's resume, identifying strengths, missing keywords and opportunities for improvement before the application is submitted.

The goal is to help candidates increase their chances of passing ATS screening while producing clearer and more targeted resumes.

---

## Features

* AI-powered resume analysis
* ATS compatibility score
* Automatic extraction of job descriptions from URLs
* Resume parsing from PDF, DOCX and TXT files
* Missing and matched keyword analysis
* Resume restructuring suggestions
* Optimized PDF export
* Secure authentication
* Personal analysis history

## Why AnaliseATS?

Most companies use Applicant Tracking Systems (ATS) to automatically screen resumes before they reach a recruiter.

AnaliseATS helps candidates tailor each application by comparing their resume against the job description and providing practical recommendations based on AI.
